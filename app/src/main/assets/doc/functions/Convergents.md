## Convergents

```
Convergents({n1, n2, ...})
```

> return the list of convergents which represents the continued fraction list `{n1, n2, ...}`.

See:
* [Wikipedia - Continued fraction](https://en.wikipedia.org/wiki/Continued_fraction)

### Examples

```
>> Convergents({2,3,4,5})
{2,7/3,30/13,157/68}

```
