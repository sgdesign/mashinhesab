## ChebyshevU

```
ChebyshevU(n, x)
```

> returns the Chebyshev polynomial of the second kind `U_n(x)`.

See:  
* [Wikipedia - Chebyshev polynomials](https://en.wikipedia.org/wiki/Chebyshev_polynomials)

### Examples
```   
>> ChebyshevU(8, x)    
1-40*x^2+240*x^4-448*x^6+256*x^8  
```
