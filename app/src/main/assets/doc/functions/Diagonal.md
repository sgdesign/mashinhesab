## Diagonal

```
Diagonal(matrix)
```

> computes the diagonal vector of the `matrix`.
  
```
Diagonal(matrix, n)
```

> computes the diagonal vector of the `n`-th diagonal above or below the main diagonal.

### Examples

```
>> Diagonal({{1,2,3},{4,5,6},{7,8,9}})
{1,5,9}

>> Diagonal({{1,2,3},{4,5,6},{7,8,9}}, -1)
{4,8}
```
