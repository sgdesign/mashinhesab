## ChebyshevT

```
ChebyshevT(n, x)
```

> returns the Chebyshev polynomial of the first kind `T_n(x)`.

See:  
* [Wikipedia - Chebyshev polynomials](https://en.wikipedia.org/wiki/Chebyshev_polynomials)

### Examples
```   
>> ChebyshevT(8, x)    
1-32*x^2+160*x^4-256*x^6+128*x^8  
```
